# Spring-boot-keycloak

Start Keycloak
From a terminal, open the keycloak-26.5.4 directory.

Enter the following command:

On Linux, run:

bin/kc.sh start-dev
On Windows, run:

bin\kc.bat start-dev

#URL admin kaycloak
http://localhost:9090/admin/master/console/#/

#URL git reference
https://github.com/UnProgramadorNaceOfficial/spring-boot-keycloak-integration

#URL keycloak configs
